# Apache MADlib (apache-madlib)
Apache MADlib is an open-source library for scalable in-database analytics. It provides data-parallel implementations of mathematical, statistical, and machine learning methods for structured and unstructured data, executed within PostgreSQL or Greenplum.

## APIs

### Apache MADlib
MADlib provides SQL-callable functions for classification, regression, clustering, dimensionality reduction, graph analytics, time series analysis, and other machine learning algorithms running directly within PostgreSQL or Greenplum.
